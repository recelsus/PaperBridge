# Panel Examples

Panel examples are grouped by profile directory.

Use `_template` when adding a new local profile. Keep uncertain values marked as
unknown in `notes.md` until they have been checked on hardware or against a
public manual/sample.

Naming guidelines:

- Use `generic_...` for controller-family starting points.
- Use `..._like` when a profile is based on public behavior but not verified as
  exact product support.
- Avoid claiming support for a finished device unless it has been tested.
